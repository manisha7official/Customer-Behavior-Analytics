# Customer Behavior Analytics & Insight Generation System

## Project Overview
This project performs exploratory data analysis (EDA) on retail customer transaction data to generate actionable business insights.

## Objectives
- Clean messy data
- Perform statistical analysis
- Generate visual insights
- Identify high-performing regions and categories

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Identified highest revenue region
- Identified top-selling category
- Calculated average purchase amount
- Generated visual sales analysis

## Conclusion
The system successfully transforms raw transaction data into meaningful insights that support business decision-making.
